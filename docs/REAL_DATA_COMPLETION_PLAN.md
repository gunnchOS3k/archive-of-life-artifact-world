# Real Data Completion Plan

This plan tracks systems that were **PARTIAL**, **MOCK_SAMPLE_ONLY**, or **BLOCKED_BY_EXTERNAL_DATA** after the Full Implementation Closure Pass, and defines how each reaches a truthful release state.

**Snapshot scope note:** Local systems target `FULLY_IMPLEMENTED for current verified snapshot scope`. Global scientific completeness requires approved source snapshot ingestion.

Machine-readable plan: `public/data/status/real_data_completion_plan.json` (generated by `npm run audit:implementation`).

## Release eligibility

- **Game-authored** mechanics (`game_authored_verified`) may ship without external snapshots.
- **Scientific coverage** requires `source_verified` provenance from imported snapshots.
- **Mock/sample** never counts as release-eligible scientific coverage.
